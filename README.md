# Kandinsk-PunktZuLinieZuFl-sche-Flaschen
Eine KI erzeugte Sammlung von Processing Scripte für Kandinskys Punkt->Linie->Fläsche Fläschen

README – Generative Kunst mit Processing
Eine Sammlung von sieben eigenständigen Processing-Skripten (Processing 4/5), die jeweils einen eigenen abstrakten Kunst-Stil erzeugen.
Jedes Skript ist komplett in sich geschlossen, läuft sofort und erzeugt per Mausklick oder automatisch immer neue Variationen.
1. eckige-polygone.pde
Mehrere unregelmäßige, eckige Polygone mit 3–12 Ecken
Transparente, bunte Füllungen → schöne Überlagerungen
Mausklick = neue Komposition
2. bezier-blobs.pde
Geschlossene, weiche organische Formen aus Bezier-Kurven
4–10 Segmente pro Blob, stark variierende Kontrollpunkte
Perfekt für flüssige, amöboide Strukturen
Mausklick = neue Komposition
3. heinz-mack-mono.pde
Klassische Heinz-Mack-Strukturreliefs der ZERO-Zeit
Schwarz-weiß-silbern, vibrierende parallele Streifen mit sinusförmigem Licht
Leichte Wellen und Neigungen → kinetische Illusion
Mausklick = neues Relief, Taste s = speichern
4. heinz-mack-farbig.pde
Bunte Weiterentwicklung von Mack (Farbchromatik-Phase)
Farbiges HSB-Spektrum, flimmernde Streifen, leuchtende Verläufe
Mausklick = neue Farbkomposition
5. organische-wurzel.pde
Eine einzige Linie wächst in Echtzeit aus einem Punkt
Perlin-Noise-gesteuerte organische Verzweigungen
Sieht aus wie Pflanzen, Neuronen, Flussdeltas oder Kalligrafie
Erreicht den Rand → neue Wurzel startet automatisch
Mausklick = manuell neu starten
6. farbentwicklung-rechts-links-unten-oben.pde
Präzises Farbfeld mit doppeltem gerichtetem Verlauf

Horizontal: von rechts nach links
Vertikal: von unten nach oben
Weicher Perlin-Noise für Wolkenstruktur
Ideal als abstrakter Hintergrund oder Farbstudie

7. mondrian-kandinsky-bezier.pde
Hybrid aus Piet Mondrian und Wassily Kandinsky
Das klassische Mondrian-Gitter besteht nicht mehr aus geraden Linien, sondern aus welligen Bezier-Kurven
Innen: lebendige, organische Farbflächen mit Kreisen und gedrehten Rechtecken
Struktur + Freiheit in einem Bild
Bedienung (bei allen Skripten gleich)

Mausklick → neue zufällige Komposition
Taste s → Bild als PNG speichern (Dateiname mit fortlaufender Nummer)
Einfach das gewünschte .pde in Processing öffnen und starten

Technische Hinweise

Processing 4 oder 5 erforderlich
Keine externen Bibliotheken nötig
Alle Skripte laufen auf Windows, macOS und Linux
Auflösung standardmäßig 1000×1000 px (einfach in size() anpassen)

Viel Spaß beim Generieren deiner eigenen kleinen digitalen Kunstsammlung!
Jedes Bild ist ein Unikat – genau wie in der echten Kunst.
